# Expatriation Framework 🛡️

Ce répertoire contient le code source du site web [Expatriation Framework](https://expatriation-framework.fr).

## Structure du site
* `intex.html` : Page d'accueil (Diagnostic & Articles)
* `vente.html` : Page de vente du Framework
* `acces-prive-framework-39.html` : Page cachée pour le téléchargement après paiement
* Les articles et pages annexes (À propos, Mentions Légales, etc.)

## Technique
* Hébergement : GitHub Pages
* Nom de domaine configuré via le fichier CNAME.
